# Comefyr Landing Page

Este proyecto es la página web oficial de **COMEFYR** (Colegio de Médicos en Medicina Familiar y Certificación de Médicos Familiares del Estado de Yucatán). Está diseñado para proporcionar información sobre cursos, eventos, membresías y la junta directiva de la asociación.

## 🚀 Tecnologías Utilizadas

- **React**: Biblioteca para construir interfaces de usuario.
- **Vite**: Herramienta de construcción rápida para el desarrollo web moderno.
- **Tailwind CSS**: Framework de CSS basado en utilidades para un diseño rápido y responsivo.
- **Lucide React**: Iconos vectoriales limpios y modernos.
- **Framer Motion**: Biblioteca para animaciones fluidas.

## 📁 Estructura del Proyecto

```text
src/
├── components/     # Componentes reutilizables (UI, Layout, Sections)
├── data/           # Archivos de configuración y contenido (textos, datos)
├── hooks/          # Hooks personalizados de React
├── utils/          # Funciones de ayuda y configuración de iconos
└── App.jsx         # Componente principal
```

## 🛠️ Instalación y Desarrollo

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/Tonnux/comefyr_landing.github.io.git
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar en entorno de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   ```
